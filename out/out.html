About

How?

Just fill out the form and print it. When you print, only the form will be visible; the nav menu and ads will disappear. Printing it to a pdf file (if your operating system allows it) and emailing that instead of using real paper is a good way to save a tree.

Is it safe?

You should be legitimately concerned about giving all this personal information to a random landlord, much less a random site. Here are some reasons this is somewhat safe, though:
  • Nothing gets sent back to any server- there's no submit button, and all the information you enter is only ever seen by your computer.
  • You can inspect the code if you're capable of doing so- this page is open source to make sure nothing devious is going on

Why?

I was surprised how hard it was to find a simple common rental application form while looking for an apartment in San Francisco, so I built this.

Who?

Kevin Kuchta - a software developer in San Francisco.

Future features:

  • Saving your progress (only on your local machine- I don't want to be responsiblefor your private information and you probably don't trust me with it anyway)
  • Perhaps having a form generator that allows realators to create their own, customized version of this form
  • Maybe more advanced features like generating the pdf for you to download or emailing it to an address of your choosing
Of course, all of this would have to be carefully thought out so-as to not interfere with the usability of the primary use-case.

Tech

This is just js- no backend. It's hosted on github pages right now, although I might move it to S3 or some other static content hosting if it gets any traffic. Libraries include the MaskedInput jquery plugin, jquery, bootstrap, and handlebars. Tools used were coffeescript, Sass, Haml, and a custom framework to run all of those that I'm working on called Canton.
